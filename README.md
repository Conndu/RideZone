# RideZone - E-commerce Motorcycle Platform

RideZone is a full-stack web application built with **Django**, designed for motorcycle enthusiasts. It features a robust inventory system, automated marketing tools, and advanced security measures.

## 🚀 Key Features
- **Complex Inventory:** Management of motorcycles, spare parts, and suppliers using a relational database.
- **Automated Marketing:** System that tracks user behavior and sends targeted email campaigns for promotions based on view counts.
- **Advanced Security:** Custom user authentication with email verification (UUID) and a suspicious login detection mechanism with automated admin alerts.
- **Dynamic Filtering:** Advanced search engine for products based on price, category, and specifications.

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** SQLite / PostgreSQL
- **DevOps:** Git for version control

## 🔧 Installation
1. Clone the repository: `git clone https://github.com/Conndu/RideZone.git`
2. Install requirements: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Start server: `python manage.py runserver`
